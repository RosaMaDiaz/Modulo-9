# Guía de Laboratorio 3 – Módulo 9  
## Consultas Multitablas y Recuperación Avanzada de Información

### 📌 Objetivo
Aplicar técnicas avanzadas de SQL para realizar consultas multitablas, optimizar la recuperación de información y comprender el uso de cláusulas y operadores en escenarios prácticos.

---

## 📌 Información General

- **Nombre de la práctica:** Consultas Multitablas, Vistas y Procedimientos Almacenados para la Recuperación Avanzada de Información
- **Modalidad:** Virtual
- **Tiempo estimado:** 4 horas
- **Materia:** Programación con JavaScript
- **Docente:** Lic. Ramón Morillo Ullola
- **Estudiante:** Rosa Ma. Díaz
  
---

### 📖 Contenido
1. **Introducción a consultas multitablas**  
   - Uso de `JOIN` (INNER, LEFT, RIGHT, FULL).  
   - Combinación de datos entre múltiples tablas.  

2. **Consultas avanzadas de información**  
   - Subconsultas (`SUBQUERY`).  
   - Operadores de conjunto (`UNION`, `INTERSECT`, `EXCEPT`).  
   - Funciones de agregación y agrupamiento (`GROUP BY`, `HAVING`).  

3. **Optimización de consultas**  
   - Índices y claves primarias/foráneas.  
   - Buenas prácticas para mejorar el rendimiento.  

4. **Ejercicios prácticos**  
   - Creación de consultas con múltiples tablas.  
   - Recuperación de información específica mediante filtros y condiciones.  
   - Aplicación de subconsultas y operadores de conjunto.  

---

### 🛠️ Requisitos
- Gestor de base de datos (MySQL, PostgreSQL, SQL Server).  
- Base de datos de ejemplo con varias tablas relacionadas.  
- Conocimientos previos de SQL básico (SELECT, WHERE, ORDER BY).  

---

### 📂 Entregables
- Script SQL con las consultas realizadas.  
- Documento con capturas de resultados y explicación breve de cada consulta.  
- Reflexión sobre la importancia de consultas multitablas en sistemas reales.  

---

### ✅ Evaluación
- Correcta implementación de `JOIN` y subconsultas.  
- Uso adecuado de funciones de agregación y operadores de conjunto.  
- Claridad en la documentación y resultados obtenidos.  

---

### ✨ Notas
Este laboratorio refuerza la capacidad de trabajar con **bases de datos relacionales** y prepara al estudiante para escenarios más complejos de análisis y gestión de información.
